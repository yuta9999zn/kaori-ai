# EU AI Act K-24 Transparency Disclosure — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Every generative AI completion carries a machine-readable AI disclosure (EU AI Act Art 50) at the llm-gateway chokepoint, and the chatbot self-identifies as AI via an SSE event. ADR-0041, invariant K-24.

**Architecture:** Add an `AiDisclosure` model + pure `build_disclosure(model_used, method)` to llm-gateway `models.py`; add a `disclosure` field to `InferResponse`, populated at its single construction site (router.py:484). Separately, the ai-orchestrator chat agent emits a `disclosure` SSE event at stream start. Purely additive — new optional field + new SSE event type; no DB, no routes, no error codes.

**Tech Stack:** Python FastAPI (llm-gateway port 8095, ai-orchestrator port 8093) · Pydantic · pytest.

**Branch:** `feat/eu-ai-act-k24-transparency` (off `main`, independent of Layer 2 / K-23).

**Invariants:** K-3 (disclosure attached at the gateway chokepoint), K-20 (model identity surfaced). No DB/tenant/route surface.

```python
class AiDisclosure(BaseModel):
    """EU AI Act Art 50 (K-24) — machine-readable AI-generated disclosure
    attached to every completion at the gateway chokepoint (K-3)."""
    generated_by_ai: bool = True
    model:           str
    method:          str   # 'internal' (on-prem) | 'external' (vendor)
    notice_vi:       str
    notice_en:       str


_DISCLOSURE_NOTICE_VI = (
    "Nội dung này do AI tạo ra. Vui lòng kiểm chứng trước khi dùng cho "
    "quyết định quan trọng."
)
_DISCLOSURE_NOTICE_EN = (
    "This content was generated by AI. Please verify before relying on it "
    "for important decisions."
)


def build_disclosure(model_used: str, method: str) -> AiDisclosure:
    """Pure builder — total, never raises. Surfaces the concrete model +
    dispatch method so the consumer (FE badge, audit) can show provenance."""
    return AiDisclosure(
        generated_by_ai=True,
        model=model_used,
        method=method,
        notice_vi=_DISCLOSURE_NOTICE_VI,
        notice_en=_DISCLOSURE_NOTICE_EN,
    )
```

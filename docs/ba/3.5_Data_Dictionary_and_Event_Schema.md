## 3. Phần còn thiếu (gap để native 3.5 đầy đủ)

- **Field-level data dictionary** (column name → semantic + PII class + retention) chưa documented per-table; phải đọc DDL.
- **Event topic schema versioning policy** (additive-only K-tenets §8): policy có, nhưng chưa registry version.
- **Lineage relationship taxonomy** (ai_decision → workflow_run → workflow_node → workflow_insight): có 12 ObjectKind enum nhưng chưa diagram.
- **Retention policy per data class** (audit 2y / workflow_event 7d / etc): rải rác trong K-1/K-6 và CLAUDE.md §7; chưa table thống nhất.
